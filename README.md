# Portfolio - Joseph Deveze

## 📋 Description
Portfolio professionnel présentant mes compétences, projets et réalisations en tant qu'étudiant en BTS SIO. Ce site web a été développé avec HTML, CSS et JavaScript.

## 🛠 Technologies Utilisées
- HTML5
- CSS3
- JavaScript
- Material Icons
- Google Fonts

## 🔍 Fonctionnalités
- Design responsive
- Navigation fluide
- Présentation des compétences techniques
- Galerie de projets professionnels et scolaires
- Section contact interactive
- Menu mobile adaptatif

## 📱 Sections
1. **Accueil** - Introduction et présentation
2. **Profil** - Présentation personnelle et parcours
3. **Compétences** - Technologies maîtrisées
4. **Réalisations Professionnelles**
   - Maison Gannac
   - Jardin Ecopaca
   - Tineesi
5. **Réalisations Scolaires**
   - Croix Rouge Business Game
   - Broceliande Immo
   - Système de Facturation
   - Application Cinema
   - Système de Ticketing
   - LiteCRM

## 🚀 Installation et Déploiement

1. Cloner le repository
```bash
git clone https://github.com/Josephdeveze/portfolio.git
```

2. Ouvrir le projet
```bash
cd portfolio
```

3. Lancer le site
- Ouvrir le fichier `index.html` dans un navigateur web
- Ou utiliser un serveur local comme Live Server dans VS Code
